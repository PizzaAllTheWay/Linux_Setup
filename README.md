# Custom Comands
Superior "ls -la" command:
```
$ lz
```
```
$ lz <DIR>
```

Superior version of "cat" command:
```
$ bat <FILE>
```

Alert comand
```
$ alert
```
```
$ sleep 5; alert "Helloooo"
```
